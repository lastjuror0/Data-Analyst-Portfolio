## Context
This folder contains an end-of-course project for **Course 4: The Power of Statistics** in the **Google Advanced Data Analytics Certificate**.

The project continues the fictional Automatidata consulting scenario and focuses on applying statistical testing to analyze data and support analytical conclusions.

## What This Project Covers
This project focuses on **statistical hypothesis testing**, rather than exploratory visualization or modeling.

It demonstrates:
- Exploring project data to support statistical analysis
- Selecting appropriate hypothesis testing methods
- Implementing statistical tests using Python
- Interpreting test results in the context of the project
- Communicating statistical findings through a concise executive summary

## Deliverables
- **PACE Strategy Document** outlining planning considerations for statistical analysis
- **Jupyter Notebook** containing hypothesis testing and supporting analysis
- **Executive Summary** reporting results and key insights

## Notes
- This is a **course-based project**
- The scenario, organizations, names, and data references are fictional
