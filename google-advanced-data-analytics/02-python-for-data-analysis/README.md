This project focuses on **data understanding and preparation using Python**, rather than modeling or advanced analysis.

It demonstrates:
- Building a dataframe from a provided dataset
- Inspecting and summarizing column data types
- Gathering basic descriptive statistics
- Organizing data in preparation for exploratory data analysis
- Answering structured questions in a guided Jupyter notebook
- Communicating progress and observations through an executive summary
