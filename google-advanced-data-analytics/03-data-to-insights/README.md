# Automatidata – Exploratory Data Analysis & Insights (Course 3)

## Context
This folder contains an end-of-course project for **Course 3: Go Beyond the Numbers – Translate Data into Insights** in the **Google Advanced Data Analytics Certificate**.

The project continues the fictional Automatidata consulting scenario and focuses on using exploratory data analysis and visualization to understand data and communicate insights.

## What This Project Covers
This project focuses on **exploratory data analysis (EDA) and insight generation**, rather than modeling.

It demonstrates:
- Performing EDA and basic data cleaning
- Selecting appropriate visualization types
- Creating plots to explore relationships between key variables
- Translating analytical findings into visuals for non-technical stakeholders
- Communicating results through an executive summary

## Deliverables
- **PACE Strategy Responses** related to Course 3 tasks
- **Jupyter Notebook** containing full exploratory data analysis
- **Tableau Visualization** highlighting key relationships between variables
- **Executive Summary** summarizing insights and including a visualization

## Notes
- This is a **course-based project**
- The scenario, organizations, names, and data references are fictional
- The dataset is created for pedagogical purposes
- Original course materials are not included
