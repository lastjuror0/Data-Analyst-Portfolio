## Context
This folder contains the **capstone project** for **Course 7** of the **Google Advanced Data Analytics Professional Certificate**.

The capstone project serves as the culminating assessment of the program and integrates skills developed across all prior courses into a single, end-to-end analytics project.

The project is based on a fictional workplace scenario and is designed to simulate the type of analytical work performed by data analytics professionals.

## Project Overview
The capstone scenario involves analyzing employee survey data for a fictional company, **Salifort Motors**, to support Human Resources decision-making.

The primary objective is to:
- Analyze factors related to employee retention
- Build a predictive model to estimate whether an employee is likely to leave the company
- Communicate findings and recommendations to non-technical stakeholders

## What This Project Covers
This project demonstrates an **end-to-end data analysis workflow**, including:
- Understanding a business problem and defining analytical objectives
- Data preparation and exploratory data analysis
- Statistical analysis and feature selection
- Building and evaluating a predictive model  
  (regression or tree-based machine learning)
- Considering ethical implications of model use
- Communicating results through a concise executive summary

## Deliverables
- **Analysis Notebook** containing data preparation, EDA, modeling, and evaluation
- **Predictive Model** estimating employee attrition
- **Visualizations** supporting analytical findings
- **Executive Summary** presenting insights and recommendations for stakeholders
