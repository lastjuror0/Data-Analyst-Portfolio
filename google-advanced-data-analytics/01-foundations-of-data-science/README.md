## Context
This folder contains an end-of-course project for **Course 1: Foundations of Data Science** in the **Google Advanced Data Analytics Certificate**.

The project is based on a fictional consulting scenario (Automatidata) and is intended to practice structuring an analytics project before any analysis or modeling begins.

## What This Project Covers
This project focuses on **project planning and data understanding**, rather than advanced analytics or modeling.

It includes:
- Completing a **PACE Strategy Document** to consider questions, tasks, and action items for each project stage
- Answering structured questions in a guided Jupyter notebook
- Performing basic coding preparation work
- Inspecting and summarizing column data types
- Communicating key findings in the form of an executive summary

## Deliverables
- **PACE Strategy Document** outlining planning decisions using the PACE framework
- **Jupyter Notebook** containing guided questions, coding preparation, and data type summaries
- **Executive Summary** communicating key observations

## Notes
- This is a **course-based project**
- The scenario, organizations, names, and data references are fictional
- Original course materials and datasets are not included
